# bux

Let's rock the world of finance.. from your terminal!


## Install
If you have [Node.js](https://nodejs.org/en/):
```
$ npm install -g bux
```

## First steps

### Login
At first, you need to log in to your BUX account:
```
$ bux login
```

### Explore
For a list of all available commands:
```
$ bux help
```
